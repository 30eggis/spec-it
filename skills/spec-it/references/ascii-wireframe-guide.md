## Tips

1. **Use consistent width**: Maintain same character width for components
2. **Express hierarchy**: Use indentation to show nesting relationships
3. **Keep it simple**: Include only essential elements
4. **Separate responsive**: Create separate Desktop / Mobile layouts
5. **Use comments**: Use `<- description` format for element explanations
