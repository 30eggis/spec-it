## Legacy ASCII Wireframe Conventions (Deprecated)

ASCII wireframe editing is deprecated. Use YAML/JSON wireframes only.
For legacy conversions, see `shared/references/common/yaml-ui-frame/10-migration.md`.

### Interactive Elements

```
[Button Text]           Clickable button
[Input Field         ]  Text input (width indicates size)
(○) Option 1            Radio button (unselected)
(●) Option 2            Radio button (selected)
[ ] Checkbox            Unchecked checkbox
[✓] Checkbox            Checked checkbox
▼ Dropdown              Select/dropdown
```

### Layout Patterns

```
# Two columns
│  Left Column    │  Right Column  │

# Card component
┌─────────────┐
│ Card Title  │
├─────────────┤
│ Content     │
└─────────────┘

# List items
│ ● Item 1                           │
│ ● Item 2                           │
│ ● Item 3                           │

# Divider with text
│  ─────────── or ────────────       │

# Tab navigation
│ [Tab 1] [Tab 2] [Tab 3]            │
│ ════════                           │
```
